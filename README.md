# Student Result Analysis using Pandas

This is a beginner-friendly Python data analysis project using the pandas library.
The project analyzes student marks and generates useful insights.

## Features
- Total and average mark calculation
- Topper identification
- Subject-wise average calculation
- Sorting students based on total marks
- Pass/Fail and grade assignment
- Ranking
- Export final results to CSV

## Technologies Used
- Python
- Pandas

## Learning Outcome
This project helped me understand:
- Data analysis using pandas
- CSV file handling
- Data filtering, sorting, and aggregation
- Writing clean Python code
